# 👩‍🍳 Chef Aanya Sharma – Culinary Portfolio Website

A warm, aesthetic, and fully responsive portfolio website built for a YouTube chef. Designed with React.js and inspired by professional culinary platforms like [Sanjeev Kapoor](https://www.sanjeevkapoor.com/) and [ChefSteps](https://www.chefsteps.com/).

## ✨ Features

- 🎨 **Warm color palette** – Terracotta, saffron, amber, and cream tones create an inviting culinary atmosphere
- 📱 **Fully responsive** – Adapts seamlessly to mobile, tablet, and desktop screens
- 🍽️ **Special Recipes section** – Showcases 6 curated signature dishes with badges, difficulty levels, and chef notes
- 📺 **Video gallery** – Displays YouTube cooking tutorials with view counts and durations
- 👤 **About section** – Highlights chef biography, awards, and experience statistics
- 🔗 **Social media integration** – 6 platform links with follower counts (YouTube, Instagram, Facebook, Twitter/X, Pinterest, TikTok)
- 📧 **Newsletter subscription** – Email capture form with success confirmation
- 🧭 **Sticky navigation** – Blurred backdrop navbar with smooth scroll and mobile hamburger menu
- ✨ **Micro-interactions** – Hover animations, scroll-triggered fade-ins, and floating UI elements
- ⚡ **Zero dependencies** – Runs entirely in the browser using React CDN and Babel standalone

## 🚀 Quick Start (No Installation Required)

This is a **single self-contained HTML file** – no build tools, npm, or server needed.

### View Locally
1. Download the `index.html` file from this repository.
2. **Double-click** the file to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. The website will render instantly with all interactive features working.

### Deploy Online (Beginner-Friendly)
| Method | Steps | Time |
|--------|-------|------|
| **Netlify Drop** | Drag & drop `index.html` onto [app.netlify.com/drop](https://app.netlify.com/drop) | 30 seconds |
| **GitHub Pages** | Push `index.html` to a public repo → Enable Pages in Settings | 2 minutes |
| **Vercel** | Deploy via [vercel.com/new](https://vercel.com/new) and upload the file | 1 minute |
| **Any static host** | Upload `index.html` to any web server (Apache, Nginx, S3, etc.) | Varies |
